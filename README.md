## 神籤 - SHIN SEN -
___
## 1. サービス概要

日常の中でちょっとした楽しみの一つとして、1日1回だけ神籤を引くことができるWebアプリです。

## 2. このアイデアはどこから生まれたか

>### 2-1. きっかけとなった体験・感情

落ち込んだり気持ちが少し沈む時に、神社で引いたおみくじの言葉に救われた経験と、

”引く瞬間のワクワク感”で気持ちを切り替えてくれたことがきっかけとなりました。

>### 2-2. なぜそれが「気になった」のか

ほんの少し前向きになることや行動のきっかけが生まれることは、事態を好転させる

きっかけとなり得ます。

そのため、「気持ちを切り替えるきっかけを、日常に組み込めないか？」と考えたことが、

このアプリの出発点です。

## 3. 課題の整理

>### 3-1. 表に見えている困りごと

- 日常が単調に感じる

- 気持ちが少し沈む日がある

- 前向きになるきっかけが少ない

>### 3-2. 本当に解決したい課題は何か

- 1回目のなぜ：
→ 単調だとモチベーションが下がるから

- 2回目のなぜ：
→ モチベーションが下がると行動量が減るから

- 3回目のなぜ：
→ 行動量が減ると自己肯定感がさらに下がるから

### 🎯 本当に解決したい課題

「気持ちの停滞防止」

このアプリは、行動そのものではなく「気持ちのスイッチ」を提供することを目的としています。

## 4. 想定ユーザーについて

>### 4-1. 想定しているユーザー
■ 年齢設定

年齢：20〜30代

理由：仕事や学習に取り組みながらも、日々のモチベーション管理に悩む年代であるため。

■ 生活・状況設定

生活・状況：勉強中・就活中・仕事中など、日々努力を重ねている人

理由：成果が見えにくい期間に、小さな前向きなきっかけを与えられる。

■ 環境設定

環境：スマートフォンを日常的に使用している環境

理由：アプリはスマホ対応しており、隙間時間に気軽に利用できる。

>### 4-2. 自分とユーザーの距離

→ 過去の自分

学習や努力の途中で、少しだけ背中を押してくれるものが欲しかった経験があるためです。

>### 4-3. 実際に使われる可能性について

最初に使ってくれそうな人：友人や家族

思い出す場面：朝の通勤時間、勉強前、気分転換したいとき

代わりに使っているもの：SNSや動画アプリ

置き換えてまで使う理由：短時間で前向きなきっかけを得られるため

>### 4-4. 導入・継続の流れ

- 学習や仕事前に「気持ちを切り替えたい」と思う

- ログインして1日1回神籤を引く

- 前向きなメッセージを受け取る

- 履歴を振り返り、自分の積み重ねを見る

「1日1回」という制限があることで、習慣化と特別感を両立しています。

## 5. 既存サービス・競合調査

>### 5-1. 似たサービス

サービス名：神社公式おみくじ

内容：参拝時におみくじを引ける

>### 5-2. それでも自分が作りたい理由

実際のおみくじは現地に行く必要があり、日常的に体験することは難しいと感じました。

また、履歴として振り返る仕組みも無いため、「日常の中で、継続的に体験できるおみくじ」

を作りたいと考えました。

>### 5-3. 差別化を一文で

このアプリは、気持ちが少し沈んだ瞬間を支えるアプリです。

## 6. このサービスで提供したい価値

>### 6-1. ユーザーの変化

行動の変化：気持ちを切り替えて行動を始めやすくなる

気持ちの変化：少し前向きになる

考え方の変化：小さなきっかけを大切にできる

>### 6-2. 価値を一文で表す

「日常に小さな前向きのきっかけを与えるサービス」

## 7. このアプリで実現すること

- ユーザー登録

- 1日1回抽選機能

- 履歴保存

- レア度別演出

- レスポンシブ対応

## 8. このアプリの懸念点とその対策

>### 懸念点

何が問題になりそうか：
一度使って終わってしまう可能性

なぜそう思うか：
機能がシンプルなため

そのために考えている対策：
UI改善による演出や分岐などゲーム性を発展、継続利用を促す

## 9. 今後の展開・発展の方向性

- 機能追加の方向性：SNS共有機能

- UI・体験の改善：演出強化

- 機能以外の工夫：PWA対応によるアプリ体験向上

## 10. 技術スタック

- DB：PostgreSQL

- デプロイ先：未定（予定：Render）
